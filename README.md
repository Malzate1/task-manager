# App Task Maker

Aplicación web para la gestión de tareas empresariales. Permite la adición, edición y eliminación de tareas para empleados de diferentes áreas de trabajo dentro de una empresa.

Cuenta con filtros para visualizar tareas:

* En progreso
* Completadas
* Pendientes

Además, requiere autenticación mediante nombre de usuario y área de trabajo correspondiente.

# Clonar repositorio

Abrir una terminal en la carpeta donde deseas descargar el proyecto y ejecutar:

```bash
git clone https://github.com/Malzate1/task-manager.git
```

---

# Instalación del proyecto

1. Abrir el proyecto en Visual Studio Code.
2. Abrir una terminal dentro del proyecto.
3. Ejecutar los siguientes comandos:

```bash
npm install
npm run dev
```

---

# Funcionalidades principales

* Crear tareas
* Editar tareas
* Eliminar tareas
* Filtrar tareas por estado
* Inicio de sesión por usuario y área
* Interfaz responsive
* Persistencia mediante API REST
